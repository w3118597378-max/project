- 📦 优化的表格布局或卡片布局
- 💫 微妙的阴影和悬停效果

### Step 1: 基础样式改造（10分钟）

在Windsurf中输入：

```
帮我改造球队排行榜页面的基础样式：

1. 移除深绿色顶部导航，替换为：
   - 1px高的橙色渐变条（from-orange-500 to-orange-400）
   - 白色页眉区域

2. 改造页眉为现代布局：
   - 左侧：橙色圆角图标容器(w-12 h-12) + 标题("球队排行榜")
   - 图标使用 Trophy，有橙色阴影(shadow-lg shadow-orange-500/25)
   - 右侧："返回首页"按钮

3. 将页面背景改为 bg-slate-50

4. 移除浅黄色搜索区背景

保持所有数据展示和功能不变。
```

### Step 4: 表格样式优化（15分钟）

**选项A: 保持表格布局（推荐用于信息密集）**

```
帮我优化数据表格样式：

1. 整体容器：
   - 白色背景(bg-white)
   - 圆角(rounded-lg)
   - 边框(border-slate-200)
   - 阴影(shadow-sm)

2. 表头区域：
   - 浅灰背景(bg-slate-50)
   - 底部边框(border-b border-slate-200)
   - 文字小号大写(text-xs font-semibold text-slate-600 uppercase)

3. 数据行：
   - 分隔线(divide-y divide-slate-100)
   - 悬停效果(hover:bg-slate-50 transition-colors)
   - 合适的padding(px-6 py-4)

4. 特殊列优化：
   - 排名列：使用橙色圆角方框突出显示第一名
   - 积分列：橙色大号字体(text-orange-600 font-bold text-lg)
   - 胜率列：使用绿色徽章(bg-green-50 text-green-600)

5. 图片优化：
   - 统一尺寸(w-12 h-12)
   - 圆角(rounded-lg)
   - 边框(border-slate-200)
```

**选项B: 改为卡片布局（推荐用于移动端或简化展示）**

```
帮我将表格改为卡片布局：

每条记录一张卡片，布局从左到右：
1. 排名数字（大号圆角方块）
2. 队徽和球衣图片
3. 球队信息（名称+教练）
4. 统计数据（场次/积分/胜率，居中对齐）
5. 状态徽章和"查看详情"按钮

卡片样式：
- 白色背景(bg-white)
- 圆角(rounded-lg)
- 边框(border-slate-200)
- 基础阴影(shadow-sm)
- 悬停效果：
  - 阴影增强(hover:shadow-md)
  - 边框变橙(hover:border-orange-300)
  - 标题变橙(group-hover:text-orange-600)
- 过渡动画(transition-all duration-200)

排名第一使用橙色突出显示。
```

   - 卡片内边距：p-6

   - 主标题：text-slate-900 font-bold
   - 正文：text-slate-700
   - 次要：text-slate-500

6. 橙色使用检查（只用于关键位置）：

   - 页眉图标
   - 搜索按钮
   - 积分数字
